# Parallel Checkmate
### 概要
ユナイテッド株式会社の3dayハッカソンにて開発したゲーム<\br>
チェスを元にした、スマホで出来るボードゲーム<\br>

### 開発環境
プランナー×1,エンジニア×2,デザイナー×1のチームで開発<\br>
開発環境はMacOS,Windowsで、言語はUinty/C#を使用<\br>
Android,iPhoneで動作確認済み<\br>
